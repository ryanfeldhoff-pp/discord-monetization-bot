# Frequently Asked Questions

## Installation

Q: What Python version is required?
A: Python 3.9 or higher

Q: Can I run on Windows?
A: Yes, the bot works on Windows, macOS, and Linux

## Usage

Q: How do I get a Discord bot token?
A: Go to Discord Developer Portal and create an application

Q: How do I change the command prefix?
A: Set BOT_PREFIX in .env file

## Troubleshooting

Q: Bot is not responding to commands
A: Check token is valid and bot has permissions

Q: Database errors
A: Ensure data/ directory exists and has write permissions

Q: Memory usage is high
A: Check for memory leaks and optimize queries
