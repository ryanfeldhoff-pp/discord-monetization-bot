# Commands Reference

## Balance Commands

### `/balance`
Check your current coin balance.

**Usage:** `/balance`

### `/balance set`
Set a user's coin balance (admin only).

**Usage:** `/balance set @user amount`

## Transaction Commands

### `/pay`
Pay coins to another user.

**Usage:** `/pay @user amount [reason]`

### `/history`
View your transaction history.

**Usage:** `/history [limit]`
